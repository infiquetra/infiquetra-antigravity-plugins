This command is an alias for `/founder-review`.

Load `saga/skills/founder-review/SKILL.md` and run the same CEO/founder-mode scope
and ambition review engine: detect the target type, run the system audit, challenge the premise,
select a committed scope mode, run the opt-in ceremony, write a `docs/founder-reviews/`
scope-decision artifact, and route in a closed loop.

It is a **review, not an implementer** — it challenges scope/ambition/direction and captures
decisions; it does **not** make code changes, commit, push, open PRs, file SDLC issues, or
`git add` anything. Keep it separate from `/strategy` (which records the direction).

`$ARGUMENTS`
